Hospital Management System

Java Swing + MySQL Project

About Project

This is a desktop-based Hospital Management System developed using Java Swing and MySQL database. It helps manage patients, employees, departments, and hospital records efficiently.

Features

  • Patient Registration
  • Employee Management
  • Department Management
  • Login System
  • Database Connectivity (JDBC)

Technologies Used

Java, Swing, MySQL, JDBC
